# gutentag-world-php

> Gutentag, World! in PHP

## Run

```
php src/GutentagWorld.php
```


## Install

```
composer require gutentag-world/gutentag-world
```


### Usage

## Usage

```sh
php src/GutentagWorld.php
```

Or require in your project:

```php
require 'vendor/autoload.php';

use function GutentagWorld\gutentag_world;

echo gutentag_world() . PHP_EOL;
```
